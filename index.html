How to use this website ?

You can go to the main view by clicking on the button

You can select the graph that you want with the dropdown

You can export the current graph with the button

You can get more information with the button

You can send a link with a direct choice for the graph file with the query
http://127.0.0.1:5500/?carto=yoyo.dot
or
http://127.0.0.1:5500/?carto=toto.dot for example.
It basically choose the option of the dropdown that match the same text
